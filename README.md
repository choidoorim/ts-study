### Controller, Service, Dao
- Controller : 요청을 받아 해당 요청에 맞는 데이터를 Service 에 주입하는 역할
- Service : 비지니스 로직을 수행한다.
- Dao(Data Access Object) : Connection 을 통해 데이터를 주고 받는 역할
